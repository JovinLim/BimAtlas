# Feature 003: Filtered Entities Table View

Popup table view of currently filtered IFC entities, opened from the main page via a Table button. Data source is the main window’s `searchState.products` (lean context over BroadcastChannel). Top segment: entity grid with default columns (Global ID, IFC Class, Name, Description, Object Type, Tag) plus user-added custom columns driven by **ENTITY** formulas (e.g. `=ENTITY.PropertySets`). Entity attributes for those columns are requested on demand from the backend (**POST /table/entity-attributes**) when the user commits a header formula. Bottom segment: **multi-sheet** area for quantity surveying (tabs, rows, columns A–F). Save/load **sheet templates** (project-scoped, name required) via toolbar search. Lock/unlock per row; protected columns (e.g. IfcClass, Global ID) are read-only. Formula bar provides active cell ref, formula input, Undo/Redo, **View full** (overlay with full cell contents, JSON pretty-printed), formula dropdown and guide. Fixture mode (`?fixture=1`) and Playwright tests allow validation without live streaming.
